📊 Detector de Patrones de Velas Japonesas (Japanese Candlestick Pattern Detector)
Este proyecto es un notebook de Python diseñado para ayudar a traders e inversores a identificar automáticamente patrones de velas japonesas en los datos de precios de acciones o criptomonedas. Los patrones de velas son una herramienta fundamental en el análisis técnico que puede indicar posibles cambios en la dirección del mercado.
El notebook simplifica el proceso al integrar la descarga de datos, la visualización de gráficos de velas y la detección de patrones específicos en un solo lugar.
🌟 Características Clave
* Descarga de Datos Automatizada: Utiliza la librería yfinance para obtener datos de precios históricos de cualquier ticker de Yahoo Finance.
* Visualización Interactiva: Genera gráficos de velas interactivas y estéticamente agradables con la librería mplfinance.
* Detección de Patrones de Candlestick: Emplea la librería TA-Lib para identificar una amplia gama de patrones de velas comunes, como Doji, Martillo, Envolvente y muchos más.
* Análisis y Resumen: Proporciona un resumen claro de los patrones detectados, indicando el tipo de patrón, la fecha en que ocurrió y una descripción de su posible implicación para el mercado.
* Fácil de Usar: El notebook está diseñado para ser ejecutado paso a paso, con celdas de código claras y comentarios explicativos.
🛠️ Tecnologías Utilizadas
* Python: El lenguaje de programación principal.
* Jupyter Notebook: El entorno de trabajo donde se ejecuta el código.
* yfinance: Para la descarga de datos financieros.
* mplfinance: Para la creación de gráficos de velas.
* TA-Lib: Para la detección de patrones técnicos (Technical Analysis Library).
